Домашняя работа №3
========================
Для запуска проекта:
1 Перейдите в директорию /hw_3/
```bash
$ cd hw_3
```

2 Запустите следующий скрипт:
```bash
$ python manage.py runserver
```

3 По умолчанию порт имеет значенин 8000, поэтому чтобы попасть на сайт, введите в адресной строке вашего браузера
127.0.0.1:8000/
